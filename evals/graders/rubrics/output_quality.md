# Output Quality Rubric

For grading agent responses to user questions.

## What to check

**Clarity**
- Easy to understand
- Technical terms explained
- Logical structure

**Completeness**
- Addresses all parts of the question
- Mentions important caveats
- User won't need immediate follow-ups

**Conciseness**
- Right length for the question
- No fluff or repetition
- Every sentence adds value

**Tone**
- Appropriate for context
- Helpful, not condescending
- Treats user as competent

**Actionability**
- Gives concrete next steps
- Uses examples effectively
- Practical, not just theory

## Score
- **1.0** - Excellent, no issues
- **0.7** - Good, minor gaps
- **0.4** - Partial, significant gaps
- **0.0** - Fails criterion
